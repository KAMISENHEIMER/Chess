# Collaboratively Created Chess Application
Focused on utilizing Object Oriented Principles, creating a clean and simple GUI, and designing machine learning opponents. 

Team 1: DROP_TABLE
* Kaden Misenheimer (rcx4)
* Danny Holt        (kid24)
* Gentry Rabe       (mot8)

Class:
Object Oriented Programming - CS 3354.003\
Fall 2024\
at Texas State University\

HOW TO COMPILE/RUN:\
> Navigate to the repository\
> Run "javac Main.java"\
> Run "java Main.java"\

Features:
- Full, working chess functionality with click controls
- "Undo" feature to rewind moves
- New game, save game, and load game
